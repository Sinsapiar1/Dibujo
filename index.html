Grabador de Dibujos

Instrucciones:

  • Iniciar Grabación: Presiona el botón para comenzar a grabar tus movimientos.
  • Dibujar: Haz clic y arrastra para dibujar en el lienzo.
  • Detener Grabación: Presiona nuevamente el botón para detener la grabación.
  • Reproducir: Visualiza tu dibujo reproduciéndose automáticamente.
  • Exportar: Descarga un archivo JSON con tus movimientos para usarlo en Python.
  • Limpiar: Borra el dibujo y comienza de nuevo.

Nota: Los dibujos exportados son compatibles con el programa Python "Bot de Dibujo".
